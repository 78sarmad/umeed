# Umeed Mobile App
A cross-platform app to help people with psychological stress. Built for https://umeed.org.pk

### Key Features
- Helpful Psychology Articles
- Book Appointment
- Audio Notes **(New)

### Languages / Frameworks
- Flutter/Dart
- PHP & MySQL based APIs
